# NLP_EngtoHindi_Transliteration-

### What is Transliteration?
    The process of converting text from one script to another script is called transliteration.
    For example, converting “namaste” to “नमस्ते”.
    (Remember that “transliteration” is different from “translation”)
    
### XML to .txt conversion
    To read the xml file I used a Python Library "xml.etree" it facilitate us to read the each element from the .xml file.
    
### RNN Model
    To solve this problem I used RNN model because it give me the facility of encoder and decoder which help me to generate new image for each words and give me
    the maximum features of that word so that at the time of testing I can get bet out of result.

### seq2seq
    On the top of the RNN I used seq2seq architecture model which we can say that based on RNN archetecture which is use for the that which have 
    some sequence i.e. like time series, and word matching.
    "A Seq2Seq model is a model that takes a sequence of items (words, letters, time series, etc) and outputs another sequence of items.
     In the case of Neural Machine Translation, the input is a series of words, and the output is the translated series of words." (Sourece: Wikipedia)
